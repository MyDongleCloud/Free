# Content of this folder
`docs` is the official documentation of MyDongle.Cloud and is leveraging [docsify](https://github.com/docsifyjs/docsify).

# Compilation of this folder
To compile this folder:
* OS requirement: **Linux Ubuntu** or similar
* Run `php ../private/modules-update.php`
* Run `./build.sh`

You will get the documentation folder `web`.
